About Me

As a passionate software and data engineering candidate, I have gained extensive experience in cloud computing, API development, data management, and monitoring during my 6-month internship. I have worked with technologies such as AWS, Python, FastAPI, SQL, Docker, Kibana, and DataHub, contributing to complex data processing and e-commerce projects. Specifically, I have focused on API integration, data analysis, and database schema design for e-commerce systems, optimizing both operational processes and customer experience. My achievements include successfully establishing data processing workflows using AWS services, developing containerized applications with Docker, monitoring and analyzing data with Kibana, optimizing data management systems with DataHub, and creating APIs for e-commerce projects using FastAPI. My goal is to specialize in data management, cloud computing, and monitoring technologies, providing innovative solutions for large-scale projects and improving business processes. With my passion for technology, eagerness to learn, and problem-solving skills, I aim to take forward steps in my software engineering career.
